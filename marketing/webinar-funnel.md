### Min 5-15: The Problem

```

"El problema no es que trabajes mucho.

Tienes dos tipos de tareas:

1. Tareas de $10/hora
   - Email, facturación, social media
   - Cualquiera puede hacerlas
   - Pero tú las haces

2. Tareas de $500/hora
   - Estrategia, partnerships, innovación
   - Solo tú puedes hacerlas
   - Pero no tienes tiempo

[SLIDE: Calculadora]

Ejemplo real:
Carlos facturaba $8K/mes, trabajaba 60h/semanas.

Pero hacía 20h/semana de tareas de $10/hora.

Costo mensual: 20h × 4.3 × $31 = $2,666
Con asistente: 20h × 4.3 × $12 = $1,032

Estaba perdiendo $1,634/mes = $19,608/año

Por hacer tareas que debería delegar.

Levanta la mano si estás haciendo tareas de $10/hora.
```

### Min 15-35: The Frameworks

```
[SLIDE: Framework 1 - Matriz de Delegación]

"Framework #1: La Matriz de Delegación

Eje X: Costo de tu hora
Eje Y: Frecuencia de la tarea

Cuadrantes:
1. Alto costo + Alta frecuencia → DELEGA PRIMERO
2. Alto costo + Baja frecuencia → Automatiza
3. Bajo costo + Alta frecuencia → Contrata asistente
4. Bajo costo + Baja frecuencia → Sigue tú

Ejercicio en vivo:
Escribe tus 3 tareas #1.

Tienen 2 minutos. [TIMING]

[SLIDE: Framework 2 - Documentar Antes]

Framework #2: Documentar ANTES de contratar

90% fallan porque contratan luego documentan.

Orden correcto:
1. Documenta el proceso (30 min)
2. Crea checklist de calidad (15 min)
3. Entrena en el proceso (60 min)
4. Delega con contexto completo

Plantilla de documentación:

PASO:
1. Abrir X software
2. Entrar a Y sección
3. Hacer Z acción
4. Verificar W criterio

DECISIÓN:
Si A sucede → Entonces B
Si C sucede → Entonces D

[SLIDE: Framework 3 - Máquina de Delegación]

Framework #3: La Máquina de Delegación

4 pasos:
1. IDENTIFICAR: Tareas repetitivas que haces >1x/semana
2. DOCUMENTAR: Proceso paso a paso con decisiones
3. ENTREGAR: Contexto completo + checklist + plazo
4. REVOLUCIONAR: Feedback estructurado, no corrección

Regla de oro:
El primer entregable estará al 70-80%.
Apruébalo.
Mejora la documentación.
El siguiente entrega al 85-90%.
Repite.

Nunca busques 100% inicial.
Busca 80% y mejora el sistema.

[SLIDE: Frameworks 4 & 5]

Framework #4: Métricas que Importan
Deja de mirar vanity metrics.
Mide lo que mueve la aguja.

- MRR (recurrentes)
- Churn (retención)
- Tu tiempo vs delegado
- Costo por tarea antes/después

Framework #5: Escalabilidad Controlada
Crecimiento = canales estables + nuevos
Nunca sumes complejidad hasta dominar simplicidad.

"
```
