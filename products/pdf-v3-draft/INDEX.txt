   $ pip3 install weasyprint

   $ pandoc CEO-Autonomo-Guia-Master.md --defaults=pandoc-config.yaml

   $ open CEO-Autonomo-Guia-Maestra-v3.pdf

⚡ COMMAND REFERENCE

Generate basic PDF:
  $ pandoc input.md --defaults=pandoc-config.yaml

Generate with custom title:
  $ pandoc input.md --defaults=pandoc-config.yaml \
      --metadata title="Custom Title"

Generate with TOC:
  $ pandoc input.md --defaults=pandoc-config.yaml --toc

Combine multiple files:
  $ pandoc ch1.md ch2.md ch3.md --defaults=pandoc-config.yaml

Show processing details:
  $ pandoc input.md --defaults=pandoc-config.yaml --verbose
